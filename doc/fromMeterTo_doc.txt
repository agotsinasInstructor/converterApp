[Γενική Περιγραφή]

Μετατροπές μέτρων σε διάφορα μεγέθη(mm, cm, km, πόδια, ιντσες)

----------------------------------------------------------------------------------------------------------------------------------

[Όνομα συνάρτησης]
MetersToMM

[Παράμετροι]
- input:        Είναι η εισαγόμενη τιμή σε μέτρα.

[Περιγραφή]
  Μετατροπή μέτρων σε χιλιοστά

[Παράδειγμα]

let input = 1;
let output = input * 1000;
Αποτέλεσμα: output = 1000 mm

----------------------------------------------------------------------------------------------------------------------------------
